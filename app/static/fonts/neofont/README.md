# NeoFont

CryoCheck carries its own local copies of the locked NeoFont branding assets:

- `NeoFont.woff2`
- `NeoFont.ttf`

The files are loaded only from CryoCheck's static asset tree. Runtime rendering
does not reference another project or any external font service.

NeoFont is opt-in branding typography. It must not be applied to body copy,
forms, buttons, tables, or general interface text.
